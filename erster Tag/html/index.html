Die Attribute des ol-Elements

  1. type
  2. start
  3. reversed

Wir können diese Attribute beliebig mischen

Type:

  1. A
  2. B

Mögliche Werte für Type

  • A - Zählt mit Großbuchstaben durch
  • a - Zählt mit Kleinbuchstaben durch
  • I - Zählt mit großgeschriebenen römischen Zahlen durch
  • i - Zählt mit kleingeschriebenen römischen Zahlen durch
  • 1 - Zählt mit arabischen Ziffern durch, ist standard

Sagt wo das Zählen beginnen soll

Die description List:

Die dl dient nicht der Aufzählung, sondern dient der Definition von Begriffen

Die dl enthält dt- description term - und dd - description detail - Elemente

Dabei ist der dt-Tag der Begriff, der beschrieben wird und dd ist die Beschreibung selbst
